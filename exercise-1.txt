   2.Tahun Kabisat :
    • Tahun yang habis dibagi 4
    •  Jika tahun habis dibagi 100, maka Tahun tersebut juga harus habis dibagi 400

    Berikut Algoritmanya :
        1. Masukkan (tahun)
        2. Jika sisa pembagian (tahun dibagi 4) != (tidak sama dengan) 0, maka cetak (“Tahun ” ,tahun, “bukan tahun kabisat”)
        3. Sebaliknya
        4. Jika sisa pembagian (tahun dibagi 100) = (sama dengan) 0 dan sisa pembagian (tahun dibagi 400) != (tidak sama dengan) 0, 
           maka cetak (“Tahun” ,tahun, “bukan tahun kabisat)
        5. sebaliknya
        6. Jika tidak memenuhi semua kondisi di atas, cetak(“Tahun” ,tahun, “adalah tahun kabisat”)

     Berikut Psudocodenya :
     Deklarasi :
       1. Input tahun;
       2. IF tahun tidak habis dibagi 4 !=0
           DO “bukan tahun kabisat”
       3. ELSE IF ((tahun %100 ==0)and (tahun%400!=0)
          DO “ Tahun Kabisat!”
       4. ELSE
	   DO”Tahun Kabisat!”
       5. END IF
       6. DISPLAY “Tahun”
	

   3.Laundry Day
    • Hitung Jumlah baju 20 kali

    Deklarasi :
    variable a
    Berikut Algoritmanya :
       1. Simpan a dengan nilai 0
       2. Ulangi a sampai level 20
       3. Tambahkan “menghitung jumlah baju” naik bersama a
       4. tampilkan  “menghitung jumlah baju”

    Berikut Psudocodenya :
       1. STORE a WITH 0
       2. WHILE a<=20
       3. Add “menghitung jumlah baju”+ a increment by 1
       4. Display “jumlah baju”
